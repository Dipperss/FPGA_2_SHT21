FPGA2SHT21_1.1.1.190412_base
是第一代基于FPGA xillinx spartan6与温湿度传感器SHT21的代码体系

采用IIC协议使FPGA与SHT21进行通信，并采用UART使FPGA向计算机进行数据传输，并由后端计算机使用python接收UART数据，并进行处理。
本套代码可以实现对温湿度的读取，数据处理要靠后端，前端收集到的数据较为不稳定，会出现较大波动，因此后端提供了大量算力用于数据滤波和计算。特别是除法计算，均在后端进行，FPGA在进行除法计算时，将会产生大量的资源占用，这是FPGA的一个弊端。目前为未找到可行的解决方案。

本套代码兼容性较差，可能对于其他方式的IIC通信不足够兼容，在alpha系列中，将实现IIC通信的模块化。
